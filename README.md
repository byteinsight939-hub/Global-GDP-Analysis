# Tableau – Global GDP Analysis Dashboard

This Tableau dashboard visualizes GDP trends & patterns across multiple countries and regions.  
It includes comparisons over time, geographic GDP distribution and country-wise GDP performance.

## Dashboard Highlights

- **World Map – GDP of All Countries**
- **GDP Belgium (2006 – 2026) Forecast Line / Bar Comparison**
- **GDP of 5 Selected Countries (2000 – 2010) – Pie Chart Summary**
- **Bhutan vs Costa Rica – GDP Comparison**
- **Scatter Plot of GDP by Countries & Year**

## Tools Used

| Tool | Purpose |
|------|---------|
| Tableau Public | Dashboard & Visualizations |
| Excel / CSV Dataset | Source data |



## Insights Observed

- GDP varies significantly by region, with stronger growth visible in some Asian countries.
- Belgium GDP exhibits both upward and downward spikes between 2006 & 2026.
- South Asian GDP contributes a notable share among selected country groups.
- Bhutan and Costa Rica show interesting GDP performance differences year-to-year.
- Scatter plot indicates how groups of countries cluster around similar GDP ranges.

## How to Run

1. Download `.twb` or `.twbx` file from this repo
2. Open using **Tableau Public Desktop Edition**
3. Connect to the dataset provided
